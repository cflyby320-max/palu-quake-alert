# Design Asset Library

Phase 4A establishes the asset library structure only. Do not place production
assets here until they have a stable ID, source note, rights note, safety review,
and `ASSET_INDEX.json` entry.

Folders:

- `backgrounds/` - textless background imagery and textures.
- `illustrations/` - textless editorial illustrations and character motifs.
- `icons/` - deterministic icon assets.
- `objects/` - reusable preparedness or civic objects.
- `patterns/` - textless patterns and motif layers.

The renderer owns all factual text, warning copy, numbers, source labels, and
the mandatory footer.
